# SEPonDashi

Premiere Pro 拡張機能「SE Pon-dashi」のインストーラーです。

---

## インストール手順

### Windows

1. このページ右上の緑の **「Code」** ボタン → **「Download ZIP」** をクリック
2. ZIPを解凍する
3. `Windows` → `SEPonDashi_Windows` フォルダを開く
4. `install.bat` を **右クリック →「管理者として実行」**
5. Premiere Pro を再起動
6. **ウィンドウ → エクステンション → SE Pon-dashi** で起動確認

### Mac

1. このページ右上の緑の **「Code」** ボタン → **「Download ZIP」** をクリック
2. ZIPを解凍する
3. `mac` → `SEPonDashi_Mac` フォルダを開く
4. `install.command` を **右クリック →「開く」**
   - 「開発元を確認できない」警告が出たら → **「開く」** をクリック
   - それでも動かない場合は、ターミナルを開いて以下を実行（コピー&ペーストでOK）：
     ```bash
     bash ~/Downloads/SEPonDashi-main/mac/SEPonDashi_Mac/install.command
     ```
5. Premiere Pro を再起動
6. **ウィンドウ → エクステンション → SEポン出し** で起動確認

---

## 更新時

同じ手順で `install.bat` を再実行するだけでOKです。  
登録済みのSEデータは保持されます。
